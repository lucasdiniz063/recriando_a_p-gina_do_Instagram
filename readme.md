# Recriando a inicial página do Instagram

Este projeto teve como objetivo recriar a página inicial de uma das maiores redes sociais do mundo: o Instagram. Contudo, como bem disse a professora no fim do curso, talvez o objetivo maior deste seja aperfeiçoar possíveis habilidades que talvez você já tenha ou trazer habilidades que podem ser desconhecidas para você, como, por exemplo, o recurso flex-box, assim como várias outros recursos que a orientadora disponibilizou durante a realização.



Este curso foi realidade na Digital Innovation One e a professora foi a Grabriela Pinheiro, Front-End Developer.
